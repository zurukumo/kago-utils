# kago-utils
Python用の麻雀ライブラリ。

Mahjong library for Python.

## インストール
```sh
pip install kago-utils
```

## 謝辞
- `tests/data/p_hon_10000.txt`
- `tests/data/p_koku_10000.txt`
- `tests/data/p_normal_10000.txt`
- `tests/data/p_tin_10000.txt`
